# Claimly
An iOS app that displays insurance claim data using mock JSONPlaceholder API.

## Features
- List and detail view for claims
- Search functionality
- MVVM architecture with Combine
- API integration using URLSession

## Setup
1. Clone the repository
2. Open `Claimly.xcodeproj` in Xcode
3. Run the project on iPhone simulator
